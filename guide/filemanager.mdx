## 🗂️ Guida Completa a FileManager

Benvenuto nella guida all'uso del **FileManager di Teledisk**, lo strumento centrale per esplorare, organizzare e interagire con i tuoi file in locale. Questa interfaccia ti permette di navigare tra migliaia di file e cartelle in modo fluido e intuitivo, con un sistema che si adatta alle tue esigenze, anche quando gestisci contenuti molto voluminosi.

### Panoramica Generale

FileManager è diviso in due aree principali:

- 📁 **Area a sinistra**: rappresenta l’**albero delle cartelle**, come su un normale computer.
- 📄 **Area a destra**: mostra i file e le cartelle contenute nella cartella selezionata.

Puoi visualizzare i file:

- 📋 In **elenco** (con nome, dimensione, tipo, data, ecc.)
- 🖼️ In **griglia** (con miniature e icone)

<Tip>
Un menu superiore ti permette di accedere rapidamente a tutte le funzioni principali: creare cartelle, ordinare i file, cercare, cambiare modalità di visualizzazione, e molto altro.
</Tip>

## Navigazione tra Cartelle

La struttura delle cartelle si comporta esattamente come quella del tuo computer:

- 🔽 Clicca su una cartella per espanderla
- 📂 Fai doppio clic per aprirla e vedere i file contenuti
- ⬅️ Usa i pulsanti **Indietro** e **Avanti** per muoverti nella cronologia
- 🔄 Usa **Aggiorna** per ricaricare il contenuto in caso di modifiche

Teledisk è ottimizzato per gestire anche centinaia di migliaia di cartelle: 👉 Le sottocartelle vengono caricate **solo quando le apri**, così da non rallentare l’interfaccia.

## Ricerca File

Hai molti file e non ricordi dove si trovano? Nessun problema.

- Digita una parola chiave nella barra di ricerca in alto
- Premi **Invio** o clicca su **Cerca**
- Visualizzerai i risultati con:


```md
 * 📄 Nome del file
 * 🗂️ Cartella in cui si trova
 * 🔗 Accesso diretto con clic
```
La ricerca è istantanea, anche su librerie di grandi dimensioni.

## Strumenti di Gestione

Per ogni file o cartella puoi:

- 🗑️ **Eliminare**
- ✏️ **Rinominare**
- 📂 **Spostare** in un’altra cartella
- ⬇️ **Scaricare** direttamente sul tuo dispositivo
- 🔗 **Generare link** o file `.strm` (per Emby/Jellyfin), se abilitato


<Tip>
Tutte le azioni sono accessibili con un clic destro sul file o cartella.
</Tip>

## Ordinamento Intelligente

Per aiutarti a mantenere tutto organizzato, FileManager offre:

- 📦 **Ordina file per estensione**: i file vengono automaticamente suddivisi in sottocartelle in base al tipo (es. .mp4, .jpg, .pdf…)
- 🎬 **Ordina per Emby/Jellyfin**:

```md
 * Modalità **Serie TV (Show)**: organizza per stagioni/episodi
 * Modalità **Film (Movie)**: organizza per nome film
```

Queste funzioni analizzano i nomi e i metadati dei file per costruire una struttura ordinata compatibile con i media server.

## Modalità di Visualizzazione

Puoi passare facilmente tra due modalità:

- 📋 **Lista**: perfetta per consultare dettagli (nome, peso, tipo, data)
- 🖼️ **Griglia**: ideale per immagini e video, mostra anteprime e icone grandi

Puoi anche scegliere la dimensione delle miniature:

- 🔹 Piccolo
- 🔸 Medio
- 🔶 Grande
<Tip>
Basta un clic per cambiare visualizzazione in base alle tue preferenze.
</Tip>

## Prestazioni Ottimizzate

Anche con milioni di file, FileManager rimane fluido perché:

- Carica solo i file e le cartelle visibili
- Usa **blocchi di caricamento** per non sovraccaricare la memoria
- Aggiorna le anteprime e le statistiche solo quando servono

Ogni cartella viene caricata in maniera “intelligente”, e se una cartella è molto grande, viene gestita a piccoli blocchi per non rallentare il sistema.

## Compatibilità con la Crittografia

Se hai attivato la **crittografia del database** in Teledisk, FileManager funziona in piena compatibilità.

- 🔒 I dati sensibili (come ID dei file) sono protetti e vengono decifrati solo al momento dell’uso
- 📸 Anche le anteprime funzionano con dati criptati, se hai inserito la password corretta
- 📂 Tutte le funzioni (spostamento, rinomina, download, ecc.) restano disponibili

In caso contrario, il sistema funziona normalmente senza crittografia attiva.

## Plugin Streaming (opzionale)

Se hai attivato il plugin di streaming locale:

- 🎞️ Puoi generare file `.strm` per integrare i tuoi file in server come **Emby** o **Jellyfin**
- 🔗 I file verranno letti direttamente da Teledisk tramite link locali

Basta fare clic destro su un file video e scegliere **"Genera STRM"**.
